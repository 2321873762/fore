## 使用方法
```
git clone https://github.com/2321873762/fore.git  //复制仓库
cd fore //进入
git branch //查看分支
git branch 名 //新建分支
git checkout 名 //切换分支

写完后添加到暂存区
git add .
git commit -m 制作人 //描述
git push https://github.com/2321873762/fore.git
